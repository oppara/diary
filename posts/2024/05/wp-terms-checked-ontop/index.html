WordPress の管理画面でチェックされたカテゴリーが上に表示されるのを抑止する

目次

問題#

WordPress の管理画面でカテゴリーをチェックすると、そのカテゴリーが上に表示されるのを止めたい。

対応方法#

add_filter('wp_terms_checklist_args', function($args) {
    $args['checked_ontop'] = false;
    return $args;
});
