GitHub Actions を設定する#

GitHub Actions による GitHub Pages への自動デプロイ - Qiita

変更箇所

  • pull_request:を削除
  • hugo のバージョンを 0.94.0 に変更
  • publish_dir: ./publicを削除
    • gh-pages ブランチの直下にデプロイする。
  • cname option を設定
    • 重要 設定しないとデプロイ時に Custom domain の設定が初期化され、URL にアクセスしても 404 になってしまう。
