GitHub Actions 時でなくてもブランチ名を利用したい

2022-04-03 · oppara
目次
  • 目的
  • 対応方法
  • 参考サイト

目的#

Actions 時に呼び出すシェルスクリプト内で、Actions 時でなくてもブランチ名を利用したい。

対応方法#

#!/bin/bash
set -Ceu

if [ "${GITHUB_REF_NAME:-UNDEF}" = "UNDEF" ]; then
  GITHUB_REF_NAME=$(git rev-parse --abbrev-ref HEAD)
fi
